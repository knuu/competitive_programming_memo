# D すぬけそだて――トレーニング――

- 解法
    - dp_{i, j}:=最後にiを使ったときに、合計j回使ったときの最大のスコアとする
    - このとき、dp_{i, j} = max(max_{0 <= k < i}(dp_{k, j - 1} + max(X, T_i - T_k)), dp_{i - 1, j})
    - ただしこれだとO(N^3)だが、Xを超える直前か、超えた直後のみを考慮すればよいため、O(N^2)に落とせる
- 感想
    - その遷移だけでよいということは直感的に分かるが証明ができてない
- 配点: 500
- タグ: DP
- レベル: 2
